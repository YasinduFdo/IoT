### &#10045; About The Project

> *This repository contains an end-to-end IoT solution encompassing hardware sensor integrations, network transport mechanisms, and a frontend user interface. It is a collaborative group project designed to capture, transmit, and visualize real-time data effectively in a structured manner.*

**Key Features:**
* **Sensor Modules:** Reliable C++ implementations for reading and managing real-time hardware data.
* **IoT Transport:** Core logic for seamless data transmission from embedded hardware to the central interface.
* **Web Dashboard:** Interactive HTML-based interface for live data visualization and system monitoring.
* **3D Unit:** Integrated 3D components and modeling resources for physical implementations.

### &#10045; Getting Started

#### Prerequisites
Ensure you have the following classic toolsets installed:
* A standard **C++ Toolchain** or Arduino IDE.
* A modern **Web Browser** to view the HTML dashboard.

#### Installation & Usage

1. **Obtain the Source Code**
   ```sh
   git clone https://github.com/YasinduFdo/IoT.git
   ```

2. **Launch the Dashboard**
   Navigate to the `Web_Dashboard` directory and open the primary `.html` file in your preferred web browser to view the interface.

3. **Configure Hardware**
   Access the C++ scripts located in `Sensor_Modules`. Update your network credentials within the code, and flash the software to your respective microcontrollers.
